# BLE Tool Box

## Platform
- Android 4.4+
- iOS 9.0+

## Sensor
Make your phone connect BLE(Bluetooth Low Energy) cycling sensors.

### Supported Sensor

- Bluetooth Heart Rate sensor
- Bluetooth Cycling and Speed Sensor
- Bluetooth Power sensor

## Android Tips:
